## Design Principles

**Typography:**
- Readability > Beauty (but we can have both!)
- Lora for long-form, Inter for UI
- 1.75 line-height (optimal for reading)
- 65ch max-width (perfect line length)

**Mobile-First:**
- Design for 375px first (iPhone SE)
- Touch targets 44px+ (48px preferred)
- Test on real devices (not just DevTools)
